## Interaction Specification

### On-Chain/Network Flows

1. **Methodology Upsert**
   - Validate required fields
   - Hash label to bytes32 ID
   - Estimate gas for transaction
   - Execute upsert with parameters
   - Show success confirmation

2. **Baseline Upsert**
   - Validate required fields
   - Hash labels to bytes32 IDs
   - Estimate gas for transaction
   - Execute upsert with parameters
   - Show success confirmation

3. **Form Management**
   - Clear forms after successful upsert
   - Show validation errors
   - Provide helpful field descriptions
