## Interaction Specification

### On-Chain/Network Flows

1. **Mint Badge**
   - Validate address, token ID, level
   - Check token ID uniqueness
   - Execute mint transaction
   - Show success confirmation

2. **Revoke Badge**
   - Validate token ID exists
   - Execute revoke transaction
   - Update badge status

3. **Adjust Reputation**
   - Validate address and amount
   - Choose increase/decrease
   - Execute reputation transaction

4. **Check Status**
   - Query badge and reputation for address
   - Display current status
